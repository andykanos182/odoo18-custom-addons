Ringkasan singkat
-----------------
- Modul ini menambahkan fitur pemindaian barcode menggunakan kamera pada field `barcode` di form `product.template`.

Tujuan & konsep
----------------
- Widget field: mendaftarkan widget field bertipe char dengan nama widget `andykanoz_barcode_scanner` (registry `fields`). View form `product.template` diubah supaya field `barcode` memakai widget ini.
- Semua asset (JS / QWeb templates / SCSS) dideklarasikan di `web.assets_backend` pada `__manifest__.py`.

Struktur file (file kunci)
-------------------------
- `__manifest__.py`: deklarasi dependensi, data dan assets.
- `__init__.py`: impor package (models kosong untuk sekarang).
- `views/product_views.xml`: mewarisi form product.template dan mengubah field `barcode` untuk memakai widget `andykanoz_barcode_scanner`.
- `static/src/js/barcode_scanner_field.js`: implementasi OWL field component (`BarcodeScannerCharField`) dan registrasi widget.
- `static/src/xml/barcode_scanner_field.xml`: template QWeb untuk field (button + modal video).
- `static/src/scss/barcode_scanner.scss`: styling modal.

Cara kerja (ringkas)
---------------------
- Field widget memanggil API `window.BarcodeDetector` untuk mendeteksi kode dari elemen `<video>`.
- Saat terdeteksi, field akan di-update via `this.props.record.update({ [this.props.name]: barcode })` dan scanner akan menutup.

Catatan kompatibilitas & risiko
------------------------------
- Barcode Detector API:
  - `window.BarcodeDetector` tidak tersedia di semua browser (umumnya Chromium-based: Chrome/Edge). Firefox/Safari mungkin tidak mendukung.
  - Jika fitur tidak tersedia, kode menampilkan pesan error; pertimbangkan polyfill atau fallback berbasis `@zxing/library` atau `jsQR`.
- Kamera & Secure Context:
  - `navigator.mediaDevices.getUserMedia()` seringkali membutuhkan secure context (HTTPS) â€” localhost biasanya diizinkan.
  - Pada perangkat tertentu/versi Chrome, fitur `BarcodeDetector` atau `torch` mungkin butuh flag eksperimental.
- Torch API:
  - `track.getCapabilities().torch` tidak tersedia di banyak perangkat; modul sudah melakukan pengecekan capability.
